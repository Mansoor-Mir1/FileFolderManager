# File & Folder Management App

A Python console application to manage files and folders.  
It supports creating, reading, editing, deleting, copying, moving, and searching files and folders in an easy-to-use menu interface.

## Features
- Create, view, edit, delete, copy, and move files
- Create, view, delete, copy, and move folders
- Search files and folders by keyword
- Logs all actions in `history.log`

## How to Run
1. Clone the repository or download the project.
2. Open terminal/command prompt and navigate to the project folder.
3. Run the program:
   ```bash
   python file_folder_manager.py
